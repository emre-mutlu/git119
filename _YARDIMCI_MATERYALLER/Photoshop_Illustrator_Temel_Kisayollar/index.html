    Photoshop & Illustrator Temel Kısayollar ve İpuçları

    Bu belge, Adobe Photoshop ve Illustrator programlarında daha hızlı ve verimli çalışmanızı sağlayacak temel kısayolları ve ipuçlarını içerir. Bu "kopya kağıdı"nı sık sık kullanarak iş akışınızı hızlandırabilirsiniz.

    1. Genel Kısayollar (Her İki Program İçin)

    • Kaydet (Save): Cmd/Ctrl + S
    • Farklı Kaydet (Save As): Cmd/Ctrl + Shift + S
    • Yeni Belge (New Document): Cmd/Ctrl + N
    • Aç (Open): Cmd/Ctrl + O
    • Kapat (Close): Cmd/Ctrl + W
    • Geri Al (Undo): Cmd/Ctrl + Z
    • İleri Al (Redo): Cmd/Ctrl + Shift + Z (Photoshop), Cmd/Ctrl + Shift + Z veya Cmd/Ctrl + Alt + Z (Illustrator, geçmiş adımları ileri alma)
    • Kes (Cut): Cmd/Ctrl + X
    • Kopyala (Copy): Cmd/Ctrl + C
    • Yapıştır (Paste): Cmd/Ctrl + V
    • Yerinde Yapıştır (Paste in Place/Front): Cmd/Ctrl + Shift + V (Illustrator), Cmd/Ctrl + Shift + Alt + V (Photoshop)
    • Tümünü Seç (Select All): Cmd/Ctrl + A
    • Seçimi Kaldır (Deselect): Cmd/Ctrl + D
    • Yakınlaştır/Uzaklaştır (Zoom In/Out): Cmd/Ctrl + (+/-) veya Alt/Option + Mouse Scroll
    • Pan (El Aracı) (Hand Tool): Spacebar tuşuna basılı tutarak

    2. Adobe Photoshop Kısayolları ve İpuçları

    2.1. Temel Araç Kısayolları

    • Move Tool: V
    • Rectangular Marquee Tool: M
    • Lasso Tool: L
    • Quick Selection Tool / Magic Wand Tool: W
    • Crop Tool: C
    • Eyedropper Tool: I
    • Spot Healing Brush Tool / Healing Brush Tool / Patch Tool: J
    • Brush Tool / Pencil Tool / Color Replacement Tool / Mixer Brush Tool: B
    • Clone Stamp Tool / Pattern Stamp Tool: S
    • Eraser Tool: E
    • Gradient Tool / Paint Bucket Tool: G
    • Pen Tool: P
    • Horizontal Type Tool: T
    • Rectangle Tool: U
    • Zoom Tool: Z
    • Default Renkler (Siyah/Beyaz): D
    • Ön Plan/Arka Plan Rengini Değiştirme: X

    2.2. Katmanlar (Layers) ve Maskeler (Masks)

    • Yeni Katman (New Layer): Cmd/Ctrl + Shift + N
    • Katman Kopyala (Duplicate Layer): Cmd/Ctrl + J
    • Katman Grubunu Oluştur (Group Layers): Cmd/Ctrl + G
    • Katman Grubunu Çöz (Ungroup Layers): Cmd/Ctrl + Shift + G
    • Layer Mask Ekle (Add Layer Mask): Katmanlar panelindeki maske ikonuna tıklayın veya Alt/Option + Katman Maskesi İkonu (gizli maske için)
    • Layer Mask'i Devre Dışı Bırak/Etkinleştir: Shift + Katman Maskesi Küçük Resmine Tıkla
    • Layer Mask'i Görüntüle: Alt/Option + Katman Maskesi Küçük Resmine Tıkla
    • Adjustment Layer Ekle: Katmanlar panelindeki Adjustment Layer ikonuna tıklayın.

    2.3. Akıllı Nesneler (Smart Objects)

    • Katmanı Akıllı Nesneye Dönüştür: Katman üzerinde sağ tıklayın > Convert to Smart Object
    • Akıllı Nesne İçeriğini Düzenle: Akıllı Nesne küçük resmine çift tıklayın.

    2.4. İpuçları

    • Maskeleme Pratiği: Keskin seçimler için Pen Tool + Vector Mask veya Select and Mask kullanın. Yumuşak geçişler için Layer Mask ve yumuşak kenarlı fırça kullanın.
    • Non-Destructive Çalışma: Her zaman Smart Object ve Adjustment Layer kullanmaya özen gösterin. Silgi (Eraser Tool) yerine Layer Mask kullanın.
    • Fırça Boyutu Değiştirme: [ ve ] tuşları.
    • Fırça Sertliği Değiştirme: Shift + [ ve Shift + ] tuşları.

    3. Adobe Illustrator Kısayolları ve İpuçları

    3.1. Temel Araç Kısayolları

    • Selection Tool: V
    • Direct Selection Tool: A
    • Pen Tool: P
    • Type Tool: T
    • Rectangle Tool (Şekil Araçları): M (Shift ile diğer şekiller arasında geçiş)
    • Elipse Tool: L
    • Eyedropper Tool: I
    • Rotate Tool: R
    • Reflect Tool: O
    • Scale Tool: S
    • Artboard Tool: Shift + O
    • Switch Fill and Stroke: X
    • Default Fill and Stroke (Beyaz Dolgu, Siyah Kontur): D

    3.2. Obje Manipülasyonları

    • Kopyala (Duplicate): Obje seçiliyken Alt/Option tuşuna basılı tutarak sürükleyin.
    • Transform Again (Tekrar Dönüştür): Cmd/Ctrl + D (Bir objeyi kopyalayıp yerleştirdikten sonra aynı işlemi tekrarlar, örn: dairesel çoğaltma)
    • Grup Oluştur (Group): Cmd/Ctrl + G
    • Grubu Çöz (Ungroup): Cmd/Ctrl + Shift + G
    • Ön Plana Getir (Bring to Front): Cmd/Ctrl + Shift + ]
    • Arkaya Gönder (Send to Back): Cmd/Ctrl + Shift + [
    • Obje Kilitle (Lock Object): Cmd/Ctrl + 2
    • Kilitli Objeyi Çöz (Unlock All Objects): Cmd/Ctrl + Alt/Option + 2
    • Pathfinder Paneli: Shift + Cmd/Ctrl + F9

    3.3. Tipografi

    • Create Outlines (Yazıyı Vektöre Çevir): Cmd/Ctrl + Shift + O (Yazıyı baskıya göndermeden veya başka bilgisayarlarda font sorunu yaşamadan önce yapmak önemlidir.)
    • Character Paneli: Cmd/Ctrl + T

    3.4. İpuçları

    • Hassas Çizim: Cmd/Ctrl + Y (Outline modu) ile vektör çizgilerini daha net görün.
    • İzometrik Görünüm: Objeleri 30, 60, 90 derecelik açılarla döndürerek izometrik illüstrasyonlar oluşturabilirsiniz.
    • Pathfinder Kullanımı: Karmaşık şekilleri oluşturmak için birkaç basit şekli birleştirme veya çıkarma işlemlerini sıkça kullanın.
    • Align Paneli: Objeleri düzenlerken her zaman hizalama panelini kullanın; göz kararı hizalamadan kaçının.

    Bu kısayollar ve ipuçları, programları daha verimli kullanmanıza yardımcı olacaktır. Pratik yaparak hızınızı artırabilirsiniz.
